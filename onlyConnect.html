À propos de « Only Connect »

Les lettres grecques sont trop prétentieuses, nous utilisons donc des hiéroglyphes égyptiens.

Ce module comporte deux phases.

Phase 1: Hiéroglyphes égyptiens

  • Le module affiche les six hiéroglyphes égyptiens habituels dans un arrangement 3×2, et un nom d'équipe.
  • Déterminez le nombre de critères auxquels chacun des hiéroglyphes égyptiens répond.
  • Sélectionnez le hiéroglyphe égyptien qui répond à un nombre de critères différents de tous les autres hiéroglyphes égyptiens.
  • Le critère n°1 est rempli si le hiéroglyphe égyptien se trouve à l'endroit spécifié.
  • Le critère nº2 est rempli si le caractère indiqué dans le numéro de série figure dans le nom de l'équipe. Si le caractère est un chiffre, convertissez-le en lettre (0=Z, 1=A, 2=B,  ;..., 9=I).
  • Le critère n°3 est rempli si la bombe possède au moins un des ports spécifiés.

  • Regroupez les 9 lettres en 3 groupes de 3, de sorte que chaque groupe contienne des lettres de la même langue. Il n'y a qu'une seule solution.
