À propos des Serrures à Combinaison

On dirait une serrure à combinaison. Je croyais désarmer cette bombe, pas la déverrouiller.

Voir l'annexe B pour la référence d'identification des piles.
Voir l'annexe Double Sécurité pour la référence d'identification des double sécurité.

Comme une serrure à combinaison typique, il faut 3 nombres pour l'ouvrir. Tourner le cadran vers la droite pour le premier nombre, ensuite vers la gauche pour le deuxième, et tourner-le vers la droite pour le dernier. Ça va l'ouvrir !

Si les nombres séquentiels dans le code sont les mêmes, effectuer un tour complet à nouveau jusqu'au même nombre.

Comment décoder la combinaison :

Chaque nombre varie de 0 à 19. Se reporter au tableau pour déterminer chaque nombre. Attention s'il y a des Double Sécurité présents car ils changent souvent !

Premier nombre :
  1. Additionner le chiffre le moins significatif de chacun des Double Sécurité. S'il n'y a pas de Double Sécurité, utiliser le dernier chiffre du numéro de série plus le nombre de modules résolus.
  2. Ajouter le nombre de piles.
  3. Si le résultat est supérieur à 19, continuer à soustraire 20 jusqu'à ce que le résultat soit compris entre 0 et 19.
Deuxième nombre :
  1. Additionner le chiffre le plus significatif de chacun des Double Sécurité. S'il n'y a pas de Double Sécurité, utiliser le nombre de modules sur la bombe (en incluant les modules Demandant)..
  2. Ajoutez le nombre de modules résolus.
  3. Si le résultat est supérieur à 19, continuer à soustraire 20 jusqu'à ce que le résultat soit compris entre 0 et 19.
Troisième nombre :
  1. Additionner les deux premiers nombres ensemble.
  2. Si le résultat est supérieur à 19, continuer à soustraire 20 jusqu'à ce que le résultat soit compris entre 0 et 19.
