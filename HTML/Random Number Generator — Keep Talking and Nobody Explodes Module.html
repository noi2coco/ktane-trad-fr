Keep Talking and Nobody Explodes Mod Random Number Generator

À propos de « Random Number Generator »

Nous sommes maintenant à la merci de RNGesus.

  • Lorsque le module s'active, il y aura deux tildes (~) sur l'afficheur.
  • Appuyez sur « Generate » pour générer des nombres aléatoires jusqu'à ce que vous trouviez un nombre qui correspond aux règles qui s'appliquent.
  • Une fois que vous avez trouvé un numéro valable, cliquez sur « Accept ».

Si le dernier chiffre du numéro de série est...

Impair : le nombre doit être impair.

Pair: le nombre doir être pair.

Si une voyelle dans le numéro de série est...

Présente : le nombre doit être compris entre 0 et 49.

Absente : le nombre doit être compris entre 50 et 99.
