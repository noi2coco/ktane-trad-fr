Keep Talking and Nobody Explodes Microcontroller

À propos de "Microcontrollers"

Ça s'appelle un "micro"-contrôleur, mais ce truc là-dedans est assez gros. Probablement parce que ça peut causer une grosse explosion...

  1. Utilisez le type imprimé du contrôleur et sa taille pour déterminer la configuration des broches à l'aide des diagrammes ci-dessous.
  2. Le point blanc sur le contrôleur indique où se trouve la broche portant le chiffre 1. Les autres broches sont dans l'ordre croissant sur le côté avec le numéro 1, et dans l'ordre décroissant de l'autre côté.
  3. À l'aide du tableau ci-dessous, déterminez le code couleur correct pour chaque élément connecté.
  4. Pour chaque broche, choisissez l'élément correct en appuyant sur les boutons HAUT et BAS et en confirmant votre saisie avec le bouton OK (la broche suivante sera sélectionnée automatiquement).
