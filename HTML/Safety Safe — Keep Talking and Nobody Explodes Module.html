À propos de "Safety Safe"

Ce coffre-fort contient d'immenses richesses ou est vide.

  • Les 6 molettes doivent être orientés correctement pour résoudre le module.
  • Chaque molette a une indication, où il clique plus fort. C'est le point de départ de chaque molette.
  • Suivez les règles ci-dessous pour déterminer jusqu'où tourner chaque molette après le point de départ.
  • Tourner le levier pour vérifier la solution. Toute molette correcte est indiquée par une lumière verte et toute molette incorrecte est indiquée par une lumière rouge.
  • A partir de 0, ajoutez le nombre de types de ports sur la bombe, multiplié par 7.
  • Ajoutez le nombre d'indicateurs allumés avec une lettre correspondante dans le numéro de série, multiplié par 5.
  • Ajoutez le nombre d'indicateurs non allumés avec une lettre correspondante dans le numéro de série.
  • Pour les cinq premières molettes, ajoutez le numéro obtenu dans le tableau de la page suivante, en utilisant l'emplacement du cadran et le numéro de série comme référence.
  • Pour la dernière molette, additionnez la somme des chiffres de la dernière colonne en utilisant tous les caractères du numéro de série comme référence.
  • Remarque : Une rotation complète prend 12 clics.
