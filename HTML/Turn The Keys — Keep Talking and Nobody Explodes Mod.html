À propos de "Turn The Keys"

L'ordre, c'est tout.

Ce module dispose de deux clés et d'un afficheur. L'afficheur indique la priorité de ce module.

CLÉ GAUCHE

Tournez la clé gauche après avoir effectué tout ce qui suit :

  • Tourner la touche droite sur tous les modules "Turn the Keys".
  • Tournre toutes les clés gauche de priorité inférieure.
  • Résoudre tous les modules "Password".
  • Résoudre tous les modules "Who's on First".
  • Résoudre tous les modules "Crazy Talk".
  • Résoudre tous les modules "Keypad".
  • Résoudre tous les modules "Listening".
  • Résoudre tous les modules "Orientation Cube".

Mais avant d'avoir fait l'une des choses suivantes :

  • Tourner toutes les clés gauche de priorité supérieure.
  • Résoudre tous les modules "Maze".
  • Résoudre tous les modules "Memory".
  • Résoudre tous les modules "Complicated Wires".
  • Résoudre tous les modules "Wire Sequence".
  • Résoudre tous les modules "Cryptography".
