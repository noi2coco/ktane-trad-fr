À propos de « Cruel Mastermind »

Le jeu de votre enfance. Seulement plus avancé. Et plus difficile à mémoriser. Et la réponse n'est plus ce qu'elle était...

  • Pour résoudre ce module, vous devez trouver le bon code de 5 couleurs.
  • Les couleurs utilisées dans ce module sont : Blanc, Magenta, Jaune, Vert, Rouge et Bleu.
  • Notez que chaque couleur peut être utilisée plusieurs fois.
  • Vous pouvez faire défiler les couleurs des DEL en cliquant sur les DEL.
  • En appuyant sur le bouton « Query », vous obtiendrez des informations sur le code actuellement entré.
  • Lorsque le bon code est entré, le module est résolu en appuyant sur le bouton « Submit ». Soumettre le mauvais code déclenchera une erreur.

Chaque requête génère deux chiffres sur l'écran. Ces nombres représentent chacun la somme totale de deux des valeurs énumérées dans le tableau ci-dessous.

A = Nombre de couleurs correctes en bonne position.
B = Nombre de couleurs correctes en mauvaise position.
C = Nombre de couleurs ne se trouvant pas dans la solution.

Couleur sur l'écran : Numéro à gauche Numéro à droite
Blanc Couleur gauche : A B
Couleur droite : # de piles # de modules résolus
Magenta Couleur gauche : C A
Couleur droite : # d'indicateurs allumés Dernier chiffre du numéro de série
Jaune Couleur gauche : B C
Couleur droite : Somme de tous les chiffres du numéro de série # de ports
Vert Couleur gauche : B A
Couleur droite : # de modules sur la bombe # d'indicateurs éteints
Rouge Couleur gauche : C B
Couleur droite : # de ports distincts # d'erreurs
Bleu Couleur gauche : A C
Couleur droite : Premier chiffre du numéro de série # de porte-piles
