Keep Talking and Nobody Explodes Mod Mastermind

À propos de « Mastermind »

Le jeu classique de décodage, avec un petit plus. Pouvez-vous vous souvenir de tout ?

  • Pour résoudre ce module, vous devez trouver le bon code de 5 couleurs.
  • Les couleurs utilisées dans ce module sont : Blanc, Magenta, Jaune, Vert, Rouge et Bleu.
  • Notez que chaque couleur peut être utilisée plusieurs fois.
  • Vous pouvez faire défiler les couleurs des DEL en cliquant sur les DEL.
  • En appuyant sur le bouton « Query », vous obtiendrez des informations sur le code actuellement entré.
    Le chiffre de gauche donne le nombre de couleurs correctes dans les bonnes positions.
    Le chiffre de droite donne le nombre de couleurs correctes dans les mauvaises positions.
  • Lorsque le bon code est entré, le module est résolu en appuyant sur le bouton « Submit ». Soumettre le mauvais code déclenchera une erreur.
