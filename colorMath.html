Keep Talking and Nobody Explodes Mod Color Math

À propos de « Color Math »

Tellement de couleurs !

  • Ce module contient 2 rangées de LEDs, à gauche (représente le numéro de base) et à droite (représente l'additionneur/soustracteur/multiplicateur/diviseur et/ou la réponse).
  • Les nombres dans ce module vont de 0 à 999999 compris. Les LEDs sont lues de haut (chiffre le plus significatif) en bas (chiffre le moins significatif).
  • Utilisez la Table 1 pour convertir les LEDs de gauche en nombre de base.
  • L'écran au milieu de ce module indique l'action à effectuer : [A]dditionner /[S]oustraire / [M]ultiplier /[D]iviser.
  • Si le texte à l'écran est VERT, utilisez la Table 2 pour convertir les LEDs de droite en additionneur/soustracteur/multiplicateur/diviseur.
  • Si le texte à l'écran est ROUGE, ignorez les LEDS de droite, puis utilisez la Table 3 pour déterminer l'additionneur/soustracteur/multiplicateur/diviseur.
  • Effectuez l'action requise et utilisez la Table 4 pour convertir la réponse en couleurs, puis saisissez-la sur les LEDs de droite. Appuyez sur « SUBMIT » pour vérifier.
